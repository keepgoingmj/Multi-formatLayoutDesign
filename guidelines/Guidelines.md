# GS 브랜드 가이드라인

## 색상

- **GS Blue** (Primary): `#0068B7`

## 로고 파일

- `src/imports/GS_logo_1.png` — 심볼 + "GS" + 구분선 + "Grow Sustainably" 풀 락업 (컬러, 밝은 배경용)
- `src/imports/GS_logo_2.png` — "GS" + 구분선 + "Grow Sustainably" 텍스트 락업 (컬러, 심볼 없음)
- **다크/GS Blue 배경**: `GS_logo_1.png`에 `filter: brightness(0) invert(1)` 적용해 흰색 반전 버전으로 사용

## 적용 규칙

- 캔버스의 카테고리 라벨, 강조 텍스트, 액센트 요소는 GS Blue를 사용
- 어두운 배경 위 소형 텍스트는 흰색 사용 (가독성)
- 탑 바 컬러 스트립, 불릿 마커, 구분선에 GS Blue 적용
- 캔버스 로고는 항상 우측 하단 (가로) 또는 우측 상단 (세로) 배치
